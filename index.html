Boguan textile · story-first references

产品图片要先讲故事再讲参数

这一版不沿用 v0.1 的行业拆解主线。重点改成：寻找那些一眼就能让客户进入场景的产品图片，再拆它们为什么能让面料变成一个可感知的商品机会。

帛冠需要的不是普通产品照，也不是网页截图，而是“故事型产品图片”：画面里必须同时出现场景、情绪、材料证据和下一步销售理由。

什么样的图片才算有故事

只拍一块布，客户只能知道“有这个材料”。故事型图片要让客户看到“这块布会进入谁的生活、什么空间、什么情绪，以及为什么值得拿样”。

1. 有人物或空间

哪怕人物很小、空间很局部，也要让客户知道这块面料会出现在哪里。

2. 有使用瞬间

坐下、打开、通勤、靠近、触摸，比平铺面料更容易触发想象。

3. 有材料近证

纹理、织法、色卡、布样边缘必须出现，否则故事会漂。

4. 有一句命名

标题不是装饰，而是把画面归纳成客户能复述的销售语言。

5. 有可落地字段

故事负责吸引，参数负责让客户相信，两个部分不能互相替代。

Reference 01 · world building

把面料放进一个客户想进入的世界

Brintons 的价值不在“图片好看”，而在于它把地毯变成了一个复古、微奢、带一点戏剧感的世界。

人物建筑纹样色卡长文案

为什么符合帛冠需求：它不是孤立展示产品，而是先让客户相信“这个主题可以成立”。面料只是故事的材料证据，但它在画面里足够大、足够清楚。

帛冠可借鉴：服装面料可以把主题换成“梅雨通勤”“长途差旅衣橱”“礼盒打开瞬间”。图片里需要有人、衣物局部、布料近景、色卡，而不是只有一张布样。

图片公式：一个可进入的情境 + 一个面料主纹理 + 3-5 个色彩编号 + 一句世界观标题。
AI 输出：标题、场景短文、材料证据字段、业务员可复述话术。

Reference 02 · product in use

让客户看到面料被使用后的样子

Pallas 的 Cirque 是更适合中小公司的方向：图片不复杂，但每张都在说明“这块布用到产品上会怎样”。

家具场景系列命名应用效果轻故事

故事性来自应用：它没有依赖大制作摄影，而是让面料出现在椅子、办公空间、软包里。客户不需要想象太多，画面已经替他完成一次“应用预演”。

帛冠可借鉴：如果做服装面料，必须出现衣服轮廓、穿着动作或人体局部；如果做家纺/软装，必须出现沙发、窗帘、床品或空间角落。产品图要从“材料照片”升级为“应用片段”。

图片公式：成品应用图 + 面料局部 + 系列小标题 + 1 句情绪解释。
适合场景：客户提案 PPT、业务员发客户预览图、样品寄送前的主题卡。

Reference 03 · tactile narrative

让一块布自己产生“触摸欲”

如果没有人物和空间，图片也不能只是平铺。Kvadrat 的做法是用折痕、阴影和纤维质感讲“柔软、自然垂坠、干净”。

触感褶皱光影极简文案

故事性来自触感：客户看见的是布料的性格，而不是一张规格图。折叠动作让材料像被人刚刚拿起，画面有“使用前一秒”的状态。

帛冠可借鉴：对基础款、素色款、功能款尤其有用。不要硬写大故事，可以让“洗后垂感”“亲肤褶皱”“轻薄透气”成为图片故事。

图片公式：单块布的折叠姿态 + 清晰纤维细节 + 留白文案 + 3 个核心参数。
适合产品：素色、棉麻、弹力、衬衫料、通勤基础款。

Reference 04 · evidence as story

把功能证据也拍成故事

功能面料不能只列检测结果。Camira 把环保来源、海岸色彩、布样组合和关键事实放到同一张卡里，让“证据”也有画面。

环保来源布样组合事实框销售证据

故事性来自转化：废弃塑料变成织物，海岸色彩变成色卡，材料来源变成销售理由。这种图片适合让采购相信“卖点是真的”。

帛冠可借鉴：抗皱、抗菌、阻燃、再生纤维、凉感、弹力这类卖点，都应该有“前因后果”的图片，而不是只把检测数字放表格里。

图片公式：来源意象 + 布样证据 + 关键数据框 + 适用场景一句话。
适合产品：功能型、环保型、制服面料、户外/差旅场景面料。

Reference 05 · small studio mood

小公司更适合做窄主题，而不是大叙事

Studio Ford 的图片说明：小型品牌不需要假装成行业巨头，只要把一个具体情绪讲完整，就会有高级感。

小型工作室生活片段自然灵感留白

故事性来自“窄”：它不是讲全品牌，而是讲一个系列的灵感和气质。图片少，但每张都服务同一个主题。

帛冠可借鉴：不用一次做十种风格。每款面料先判断它最适合哪个微型世界：雨天通勤、春夏衬衫、旅行箱、酒店房间、童装礼盒、户外午后。

图片公式：2 张生活片段 + 1 个材料近景 + 一段灵感来源。
适合产品：季节小系列、花型款、联名款、客户定制主题。

Reference 06 · named scene

用地点、时刻、动作给系列命名

故事型图片最怕泛泛而谈。“高级”“舒适”“自然”都太空，地点和动作更容易让客户记住。

地点叙事旅行感系列命名PPT开场

故事性来自命名：DESTINATIONS 把每组产品放进地点线索里，图片天然适合做提案开场。

帛冠可借鉴：AI 不只生成“产品名”，还要生成“客户能理解的场景名”。例如：梅雨通勤、候机厅午后、商务酒店床尾、春季样衣间。

图片公式：场景名 + 情绪图 + 材料图 + 适用客户。
适合输出：客户提案 PPT 第一页、企划卡封面、业务员朋友圈/私域预热图。

仍然需要证据，但不能让证据抢走故事

规格页、Line Sheet、色卡页不是主角。它们应该作为故事图片后的第二层，用来回答客户“能不能落地、能不能采购、能不能打样”。

下一版不要再做这三类主图

它们不是不能出现，而是不能作为第一眼的主视觉。第一眼必须先把客户带进故事。

不要用产品网格当主图

产品网格适合选货，不适合激发客户想象。它只能放在故事图之后。

不要用孤立布样当主图

布样只能证明材质存在，不能说明它会出现在哪个生活或商业场景。

不要让规格表开场

参数是信任层，不是吸引层。客户先被画面吸引，才愿意看参数。

帛冠 V1 的图片方向应该改成“故事主图 + 证据副图”

每张产品卡先生成一张故事型主图：有具体场景、有使用瞬间、有材料近证、有一句可复述的主题名。再生成一张证据副图：色卡、规格、性能、MOQ、交期和打样动作。这样才既符合小型公司的实际生产能力，也能让业务员拿去和客户沟通。

建议下一步：围绕帛冠真实面料，先做 4 个故事主题样张，而不是继续扩展案例数量。
